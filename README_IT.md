# TestLab

## Overview
The TestLab library provides an easy-to-use API for test lab functionalities. It is designed to simplify the management of test data, database operations, and FRF processing.

## Installation
To install the library, use the following command:

```bash
pip install TestLab
```

## Usage
### Basic Usage
```python
from testlab_easy.testlab import TestLab

# Create a TestLab instance
lab = TestLab()
```

### FRF Processing
```python
from examples.frf_processing import process_frf

process_frf(...)
```
